
# Martix_Multiplication with cuda
The examples are written for Linux, not Windows.

# How-to

## For test_cuda.cu
### Compilation
The simplest way to configure is to run command:
```
nvcc -o test_cuda test_cuda.cu 
```

### Run
The following command runs the compiled codes.

```
./test_cuda
```
